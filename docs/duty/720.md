# 伊甸零式希望乐园 共鸣之章2

基本站位中，根据打法不同，D1/D3、D2/D4可能发生互换，请队内确认。
```
/p ■基本站位
/p 　MT/D3　　ST/D4
/p 　H1/D1　　H2/D2
/p ■火接线　远程双H 
/p ■双体　　左：STH1D3D4 / 右：MTH2D1D2
/p ■切割　　死刑１点 　　※切割左/上集合
/p ■火龙卷(同向式)TH→西、DPS→东
/p            MT/H
/p 　D1D3　D2D4
/p             ST/H　　　※同侧D3D2换
/p ■合体接线
/p 　先H1MTSTH2 / 后D3D1D2D4
/p 　※四角分摊后TH不动，D顺时针移动45度 
```
