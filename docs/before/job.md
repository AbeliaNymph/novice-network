# 战斗职业介绍

估计这个页面要额外设计 UI

[战斗职业练级手法解说(全)]T
FF14的战斗职业系统比较特殊，你可以用一个角色畅玩所有职业，全职业满级大佬也遍地都是。因此不用太过于纠结职业，玩的不爽换一个就是。
FF14的战斗职业分为基础职业和特职，创建新账号时所选择的是基础职业，而我们通常所说的职业(骑士、龙骑等等)都是指的特职。当你的基础职业达到30级时，职业行会导师就会带你去找你的特职导师了~
在一般来说，转职前的基础职业和转职后会被视为同一职业(因为经验、等级和加点都是共通的)，下文小标题中会一并注明特职和对应的基础职业，方便你在进入游戏时选择。

游戏在2.4中，追加了忍者；在3.0中，追加了暗黑骑士、占星术士和机工士；在4.0中，追加了武士和赤魔法师。
创建新账号时都无法选择上述这几个职业，忍者(双剑师)可以在游戏初期转职，剩下的职业都要求你有任意一个50级的战斗职业，并且完成对应的主线任务才可以。

FF14的职业分工与其他MMORPG很相似，都是T(坦)奶DPS分工的制度。一般副本中，采取1T1奶2DPS的小队组合；而高难度副本中，采用8人小队组合，需要2T、2奶和4个DPS。
默认分配中，4个DPS为2近战，2远程。虽然也有3近战+1物远的菜刀队组合，但是综合输出高于2近2远的情况并不多见。
FF14的职业平衡做的相对较为良好，在挑战大型困难副本时，基本不存在哪个职业失业的情况，更重要的参考是玩家的学习能力、反应能力和对职业的理解与操作。所以在选择职业时，要以自己的能力和喜好为主。

下面的简要介绍，我尽量提供主攻随机日常周常的休闲玩家和主攻8人RAID级副本的挑战玩家两种视角的见解，但是职业具体技能请移步各个职业攻略(版头都有传送门)。
以下内容包含本人主观看法，接受各位大神友好探讨(……
坦克/Tank
T在副本中负责拉住怪，保护其他队友，掌控副本节奏，带领团队走向胜利，如同团队的领导者一般。因此T需要十分熟悉各个副本的路线、机制和应对方法，优秀的T更是能够把握团队每名成员的特性，并以此安排攻略节奏。
在4人本中，T需要熟悉每个职业的输出模式，认识近战怪和远程怪以及瞎子怪，选择合适的拉怪方式，而瞎子怪更可以绕过不打。需要认识各个DPS和奶妈的场地型技能，把怪或者自己放在场地技能的圆圈中。还需要能够识别怪的方位，为近战提供输出环境。
− 什么是输出环境？ ...
在副本里，T请不要围着怪转圈，拉好怪站稳后，在打完之前请不要长距离移动，因为这样会导致近战损失海量输出(近战输出是需要打怪的特定方向的)，继而降低你们团队的输出和通关速度。
在8人副本中，则需要2个T进行配合，这两个T分别称作MT(主坦)和ST(副坦)，MT承担绝大多数的BOSS伤害并提供团队保护，ST则承担大部分的机制处理职责。MT与ST的职责有可能发生交换(换T)，某些情况下MT和ST的职责几乎没有区别，仅仅是谁先上的问题而已。他们相对其他职业更加集中于减伤的管理，部分副本要求T有良好的带位，以应对后续的副本机制。
FF14到目前为止都没有“废”副本，如果要排每日随机的话，任何副本都有可能进去，这就对T产生了比较高的要求(直升玩家需格外注意)。

强烈不建议没有任何MMORPG经验的玩家以T作为自己的第一个职业，除非你对学习能力、应变能力和抗压能力十分自信。作为团队的领导者，T的工作复杂而沉重，在一般的4人副本中，DPS甚至治疗躺了都还有机会打过BOSS，而如果T躺了几乎就是要从头再来了。
而拉不住怪、不会开减伤、瞎转圈or瞎走位、不会处理机制、不会带位等等，都有可能遭到不友好的对待(轻则默默拉黑被票出本，重则开喷表人从而都有可能)，在8人高难度副本中，不能履行职责的T会被视为不合格，可能会被踢出团队。
WOW及其他游戏的T玩家需要注意的是，ff14的T与其他T有一些细节上的不同，比如仇恨机制和减伤管理方式…具体内容可以下载后面的电子书(章节副本攻略)或者自行体会…[一图流简易职业攻略]T，[防护职业攻略]T。

当然另一方面，T本身的乐趣与成就感也来源于此，对全局的观察与把控，对自我的要求与提升都是T玩家不断精进的动力，如果你想要快速了解这个游戏的战斗细节，精通T职业本身也将会是一条有效的捷径。
骑士 (剑术师)
− 骑士介绍 ...
剑术师原本是乌尔达哈的斗技场中使用剑盾生存到最后的斗士，成为骑士之后就成为了守卫乌尔达哈王族的银胄团精英，使用剑术与盾牌进行攻防，保护自己的君主和效忠的王国。
骑士是最传统的T职业，操作相对较为简单，减伤丰富，并且拥有真正的无敌，也比较适合初次接触T职业的玩家学习。
60级之前的骑士输出手段比较简单，让玩家可以有更多精力放在减伤走位上，同时他作为一个可以加血并且拥有团辅能力的T，在应急处理上也比另外两个T更加游刃有余。60级之后追加了魔法输出技能，让骑士T在不失去原有优势的情况下大幅度加强了输出能力。现在的优秀骑士T在MT位时，能够为团队提供最完美的输出空间，并在出现异常紧急情况的时候帮助奶妈分担一定的压力。在ST时能打出可观的输出，保护全团的同时为团队输出锦上添花。
在目前大型RAID副本中，骑士更加偏向于ST位，由于骑士MT位输出损失严重，在DPS要求非常严格且需要换T的副本中，出场率也会随之下降。
推荐初次挑战T职业，或有骑士(以及圣骑士)情结的玩家选择。

战士 (斧术师)
− 战士介绍 ...
斧术师是生活在拉诺西亚大海上的海贼，接受了狂暴的力量而成为战士。
战士曾经被笑称为“只是血比较多的DPS”。他拥有冷却时间最短的小减伤“原初之魂”，这个技能使得他成为了T5(2.0版本50级Raid副本)的MT首选，虽然之后战士一直作为ST爆炸输出，但从这个版本开始，随着骑士转移到了ST位，战士也开始在MT位上绽放新的 我特码锯爆(换着姿势锯爆(翻着花儿锯爆 。
战士的类似无敌技能为死斗，使用死斗之后，血量无法降低到1血以下，需要另外一个T配合嘲讽，以及奶妈及时抬血，否则死斗时间结束后，只剩下个死战士。
战士拥有3个T中最强的自奶能力，野外小怪SOLO中可以完全无伤而退，这导致战士经常需要承担一些大伤害技能，而且很有可能莫名被奶妈放生。但另外一方面，战士的防御能力是3个T之中最差的，战士的T姿态仅仅是增加了最大血量和接受的治疗量，而战士除了标配的大小减伤之外，都是回血/增加自身血量上限的技能，所以如果没有较好的减伤意识和操作，战士暴毙的几率远高于另外两个T。
但是一个优秀的战士T在4人副本中能展现出无可比拟的战斗力，输出高仇恨稳，还能花样自奶。在8人副本中，战士要根据搭档的职业变更自己的职位，因此相对来说更吃意识和操作，尤其是在野队里，战士对副本整体的节奏和双T需要完成的工作与职责都需要了如指掌，才能更好地打出自己应有的输出水平。
推荐想要玩T，又想要超高输出的玩家选择。

暗黑骑士
− 暗黑骑士介绍 ...
暗黑骑士是伊修加德背负罪孽的骑士，他们堕入黑暗而不再使用盾牌，他们为穷苦的百姓而战，为了心中的正义而战。
黑骑是3.0追加的新特职，曾以复杂的蓝量监控和琐碎的技能著称。而暗黑骑士挨打回复战斗资源的特性，和极强的魔法减伤能力，让暗黑骑士成为了整个3.X大版本中大多数副本当仁不让的MT。
暗黑骑士的类似无敌技能是行尸走肉，当黑骑开启行尸走肉之后，在buff时间内受到致死伤害时，血量被锁为1，并且buff变为“死而不僵”( 死男人 )状态，此时需要奶妈为黑骑加血。当治疗量总和大于黑骑血量上限时，即可解除死而不僵状态，否则10s后直接暴毙(死透了…)。这个技能通常和白魔的天赐绑定，在没有白魔的时候，只能通过占星的先天和学者的活性快速抬血，暴毙概率较高。
暗黑骑士的暗黑buff能够抵消盾姿带来的伤害降低，使得暗黑骑士在拥有盾姿的防御加成的前提下，几乎不会降输出。同时在关盾的情况下，更能带来20%的伤害加成。追求这20%加成的暗黑骑士，会在大部分副本中选择关盾输出。
在当前版本中，4人副本的黑骑相对来说较为简单，输出也十分不俗。而在高难副本中，由于黑骑钦定MT位保持不变，如何管理减伤和仇恨都是挑战高难度副本的黑骑必须慎重考虑的事情。
推荐追求帅气(划掉)，尝试更复杂操作的玩家选择。
治疗/Healer
故名思义就是奶妈，负责给掉血的T 和DPS 提供抬血服务。奶妈需要认得很多buff图标，针对不同的buff采取不同的处理措施(比如骑士无敌时可以输出一波，而黑骑死男人状态时及时抬血等等)。在某些场合，奶妈必须及时驱散负面buff，否则轻则团队掉血降DPS，重则直接灭团(4.0中对可驱散debuff做了标志，可驱散的buff图标上方会有一条小小的线←笑容遇到魔界花)。
FF14中的奶妈最不同于其他MMORPG的奶妈的一点就是，他居然需要输出！是的，首先，治疗完全有能力单独完成主线任务，这就要求他有一定的输出能力，在设计上，治疗的输出能力也是非常不错的 限制我DPS的只有队友和蓝量 。其次，FF14治疗技能大都奶量不错，因而能给治疗带来不少“发呆时间”，在难度较高的8人副本中，治疗就需要利用这些“发呆时间”来辅助输出，在比较好的团队中，奶妈经常需要承担大约半个DPS的输出量(两个治疗加起来)。
因此在副本中积极治疗之余，也来尝试一下输出吧~(但可千万不要顾此失彼，为了输出而忘了加血，那可是会团灭的)。
白魔法师 (幻术师)
− 白魔法师介绍 ...
白魔法使用来自黑衣森林元灵的力量，特点是群奶和HOT(持续性治疗)，并且可以使用土、风系的攻击魔法。白魔法师作为3个奶中最好上手的治疗职业，非常符合一般玩家掉血→加血的思考过程，也非常适合新人学习奶妈操作。
4人副本中的白魔奶量贼大，抬满团血轻松简单，一般DPS掉血只要瞬发一个再生等他慢慢回满就行了。而输出白魔更是化身人形投石机或者瞎眼爆弹，不把蓝砸光不罢休。50级时代最晃瞎眼的技能，也是来自白魔的神圣。
在8人高难度副本中，白魔以超强的群体治疗能力主导团队整体血线水平，在一定程度上照顾T血，甚至一部分应急处理也都需要白魔来做(因为白魔瞬发能力技较多)。但是白魔的HOT技能让他在团队稳定输出的时候，也能添上自己的一份力。而新手白魔和浪输出白魔最常发生的情况就是——没蓝…白魔是最早出现控蓝问题的奶妈，如何控蓝也贯穿白魔法师前60级的人生。
4.0的白魔法师出现了12s不耗蓝的技能，导致白魔法师操作的最后一堵屏障也轰然倒塌，无论是输出还是加血都可以更无脑(所谓胸大无脑啦)。整体来说开荒队伍都会选择带上一个白魔保证团队血量容错，并提供其他二奶望尘莫及的输出，而刚开始接触奶妈的玩家也可以尝试从白魔入手。

学者 (秘术师)
− 学者介绍 ...
学者是尼姆的守卫者，如今作为海都海兵的治疗分队，与自己召唤的小仙女并肩战斗。他的特点是拥有极强的减伤能力，并且能够使用秘术师的攻击能力。
学者的各种护盾、防护罩都需要学者对副本机制有较强的理解和记忆，学者必须能够在伤害来临之前布置好所有的减伤，否则减伤无效不说，还有可能导致灭团。
小仙女作为不占公共CD的持续奶，在一般的时候会自动选择小队中血量低于80%的队员进行治疗，在学者的手动操作下，也可以对某一特定目标进行持续治疗。大部分学者仗着小仙女的加持和护盾的伤害吸收能力，输出起来肆无忌惮，也因此得到了气炎法师的美誉。学者曾经也拥有最强的瞬间抬血能力(生命活性法)，在3.0之后，抬血能力遭到削弱，而且在大多数场合，学者并不适合持续性大规模的团队抬血。
4人副本的学者并不会时刻把团队所有成员的血抬满，而是有选择地为队员保持鼓舞盾，配合小仙女的持续治疗，从而让团血保持在较高的位置，而自身又可以游刃有余地输出。
大型高难副本中的学者，通常只是布置必要减伤，以及在主奶不能独自抬够血量时搭把手。优秀的学者可以在完成治疗任务之余打出极高的输出。同时只要对该职业和副本拥有足够理解，充分发挥他的治疗能力，学者的表现并不会逊于另外两个奶妈。
推荐有盾奶经验，以及想要尝试治疗输出的玩家选择。

占星术士
− 占星术士介绍 ...
占星术士是3.0的追加特职，来自萨雷安的智者从星空中得到力量和指引，使用星星的力量进行攻击和治疗，在伊修加德 推销 推广这种拥有治愈能力的占星术。占星术士拥有白昼和黑夜两个形态，切换不同的形态，会得到不同的属性加成和技能效果。同时占星拥有“奥秘卡”系统，通过抽卡和使用卡牌，为团队带来各种各样的增益。同时也因为奥秘卡系统，导致占星成为了操作最复杂的奶妈没有之一。
占星的白昼学派类似于白魔，拥有HOT能力，而黑夜学派则像学者，拥有护盾能力。在4人副本中，占星和其他奶妈没有太大区别，只是他们需要花更多GCD打牌和加血(而非输出)。
而在8人高难度副本中，占星需要根据团队中的另外一个奶妈的类型，决定自己的学派种类(有白魔开夜占，有学者开白占，双占星协调切职业…)。而6种不同的奥秘卡则拥有不同的能力，烧掉奥秘卡也可以获得额外的卡牌增益。所有这些机制都要求占星对副本机制和队友特点有详细的了解，在适当的场合给出适当的卡牌，如何烧卡给牌，一直都是占星玩家致力研究的课题(沉迷打牌)。
因为占星是个偏辅助型的奶妈，而奶量并不能完全与另外两个奶妈分庭抗礼，输出更是低的不行，所以占星玩家需要更多经验和更细致的手法，同时一位优秀的占星又是团队不可多得的强力成员。
推荐爱美，或想要帮助团队打出更高输出的玩家选择。
进攻职业/DPS
在4.0中，DPS被划分为近战物理、远程物理和远程魔法3种类型。下面也以此为顺序进行介绍。
DPS就是在副本中攻击BOSS的主力，躲开技能、处理各种机制和正确打出输出循环是所有DPS的基本功。配合副本机制调整输出循环和手法也是一个进阶DPS需要认真考虑的事情。
在最近大半年的T奶输出热潮下，很多DPS将通关速度慢、没能压掉阶段等等问题怪罪于T奶。这完全是舍本逐末的行为。
请记住，如果你们打得慢了，99%都是DPS的输出问题，剩下1%的问题可能来源于 土豆服务器 ，或者家长拔网线，猫咪踩键盘。
FF14副本中的BOSS有各种各样的攻击方式，单点某一个DPS的集合、分散，吹飞、石牢/水泡/锁链等等，各种各样的机制也是FF14副本的乐趣所在，恕我在这里实在没办法讲全。
在8人高难度副本中，很多技能机制需要所有DPS乃至所有成员共同处理，如果一个人出错就有可能导致团灭，这种紧张和惊心动魄的心情，是FF14副本挑战的魅力。

近战物理
近战物理职业，顾名思义是贴近BOSS进行近战物理输出的职业。如果你喜欢贴近boss(有身材爆炸的小姐姐，也有一身脓包的…恶……)的刺激感，那么近战一定是你的心头好。通常来说副本中都是近战伤害最高，然而这些职业的技能或多或少都有身位要求(也就是技能要求在BOSS的背面、侧面使用)，如果不能在正确的身位使用技能，伤害量就会大幅下降，无法成为合格的近战DPS。
在8人高难度副本中，至少需要带上2名近战DPS。
龙骑士 (枪术师)
− 龙骑士介绍 ...
长枪是遍布艾欧泽亚的主要武骑，枪术士行会所在的黑衣森林的住民就使用长枪狩猎，而伊修加德的龙骑士则使用长枪发展出在空中与龙交战的战技。
龙骑士是3.0的主角，苍天的伊修加德，苍天的龙骑士，整个龙诗战争就是人与龙之间的斗争。作为曾经的主角强了一个版本，在4.0预告片中有个NPC用长棍/枪疏通大炮的镜头让大家纷纷感慨，龙骑沦为了擦大炮的。
龙骑的主要输出手段是保持龙血，配合蹦跳技能和常规5连击开启红龙血，并打出红龙炮。因为这是在3个版本中逐渐开放的功能，所以50级、60级和70级龙骑会拥有3种不同的输出手法。
相对于其他3个近战物理职业来说，龙骑机动性比较好，但他的技能有一定的身位要求，否则也无法打出完美的循环。龙骑的缺点同样来自于其机动性，因为他的所有蹦跳类技能的硬直动画都极长，导致不熟悉副本时间轴的龙骑(还有瞎78浪的龙骑)经常会在脚下出现BOSS攻击圈的时候，因为技能动画僵直而无法移动…然后惨惨地躺在地板上(或者浪操作一个后跳飞身坠落)。这也是龙骑经常被戏称为放LB的躺尸龙的主要原因之一。
龙骑因其buff对齐方法多样，开场爆发难以与团队爆发对齐，以及相对较大的死亡惩罚，导致龙骑玩家输出的上下限差距较大。但是帅啊，另外一方面，龙骑也是诗人机工的好基友，在龙骑的加持之下，物远职业甚至有能力打出堪比武士的输出。

武僧 (格斗家)
− 武僧介绍 ...
在乌尔达哈斗技场中生存的拳术师，掌握并开启脉轮就能够成为武僧。4.0的主角，阿拉米格的拯救者。虽然是新一代主角但并未得到所谓史诗般加强，倒是2.0延续下来的6个技能打一套核心没有任何变化。
武僧的主要输出方式是使用技能在3种身形之间切换，魔猿→盗龙→猛豹→魔猿→…，每个姿势对应2个技能，每成功打出一次三连(就是打出豹猛身形时的技能)就会叠加一层疾风迅雷，3层疾风迅雷就是武僧的常驻buff。虽然一共只有6个核心技能，但是每个技能都有自己的身位要求，而且分别都有对应的buff…所以武僧打起来如同猴子一般，围着boss没完没了地乱转。如同职业导师所言，优秀的武僧打出一套技能有如行云流水一般，一点一滴打出最强伤害。
武僧早期是木桩之王的角色(忍者黑魔沉了又浮)，但是这是建立在目标不上天、不位移、不转圈的前提下。武僧一但断了疾风迅雷，就如同断臂的 杨过 维纳斯一样废的不行，在后续的版本中，陆续为武僧增加了利用疾风迅雷的方法和技能。如今的武僧只要管理好疾风迅雷和斗气，并将其对齐各种buff技能即可。虽然对于新人来说，武僧可能是最为复杂的近战职业，但是对于老武僧来说，长年换汤不换药的打法可是有点无聊了。
虽然武士一度威胁到了武僧的地位，但是伴随着4.2版本的武僧加强，封面职业的辉煌又再次降临。武僧拥有近战中最强的打击感，也有自己的专属武器(武僧是唯一一个可以“空手”打怪的职业)，是热衷近战肉搏玩家的上佳之选。

忍者 (双剑师)
− 忍者介绍 ...
忍者是2.4新增的职业，从远东流亡而来的忍者在利姆萨罗敏萨落脚，带来的新职业体系，当年很多躺尸龙迫不得已都练了一个忍者以求顺利通过巴哈姆特大迷宫…
忍者的特点是通过“天”“地”“人”三种印结印，从而使出6种不同的忍术(如果加上兔兔和手里剑就是8种…嗯！)。但是因为结印需要在1个GCD里连按3~4个技能，导致忍者 苦练钢琴 成为了一个不是那么轻松容易上手的职业。在早期忍者是极强的单体输出，如今的忍者是一个带有大量团辅的高输出职业。
忍者在保证风遁不断的前提下，使用忍术和普通攻击技能输出，配合普攻提供的忍气进一步加强忍术的使用。每60s一次的背刺(攻其不备)是全团都可享受的极强增伤，也是忍者本体的重要爆发技能。一般来说挑战高难副本的固定团都会选择家养忍者，并逐渐配合出固定的背刺时机。而最优秀的忍者可以凭经验判断出背刺的时机，为自身和团队带去更高的输出。而高难度团本中，忍者几乎是必备成员，因为他的仇恨控制技能，可以帮助T和DPS稳定自身仇恨，输出更加肆无忌惮。
忍者自带的辅助类特性和技能是最为丰富的，比如自带移动速度加快，隐身在PVP中可以偷点，缩地是唯一可以指定落点的位移技能，影渡和烟雾弹分别能够转移自身仇恨，抑制目标仇恨上涨等等。
如果你觉得2.5s GCD太漫长，那么练满忍者可以让你后悔自己的无知与年少。

 武士
− 武士介绍 ...
4.0新增的物理型近战职业，只要玩家拥有任意一个50级战斗职业就可以去乌尔达哈转职成为武士。
武士使用太刀通过连击打出“雪”“月”“花”三种闪，消耗闪或者剑气打出更强力的输出。武士就是新一代的自嗨王，他几乎不提供任何团队辅助(雪闪虽然带斩耐，但通常由战士和忍者负责)，仅靠单体打出强力的输出。
武士本身的基本输出方式简单又暴力，而进阶的武士又需要通过各种能力技管理剑气，继而调整输出和爆发节奏，无论是初次接触武士还是追求更高更强的武士，都能找到自己的钻研重点。
在目前版本中，装等接近的情况下，如果武士玩家没有打出团队最高输出，那你就完全浪费这个职业了。

远程物理
远程物理使用弓箭或火枪进行远程物理攻击，虽然最早原本定位上是团队辅助，但其实他们已经成为不折不扣的输出职业了。
如果你不喜欢挨着boss站，又不喜欢磨磨唧唧的读条，物远就是你最好的选择。
吟游诗人 (弓箭手)
− 吟游诗人介绍 ...
黑衣森林的猎手使用弓箭袭击它们的猎物，而学会了各种战歌的弓箭手就成为了吟游诗人。诗人使用弓箭作为进攻手段，用战歌为团队辅助。在2.X版本时代，诗人的LB和奶妈效果是一样的(回血/复活)，导致在3.0之初诗人拥有了自己的专属LB之后，诗人抢放LB的事件频频发生。
现在诗人的输出，是使用3首不同的歌曲及其带来不同的效果，为团队带来稳定的2%暴击增益，并打出自身的最高伤害。同时诗人也变成了游戏中要监控自身buff最多的职业。优秀的诗人需要能够合理安排三首歌的演唱时间和顺序，并在对应的歌曲中打出对应的技能，同时能够配合团队爆发节奏提供增益，并为奶妈提供减伤帮助。如果你喜欢多线程思考，诗人算是一个比较优良的选择。短时间的buff监视，长时间的技能规划都需要细致考虑，才能打得更好。
作为艾欧泽亚钦定爱豆，诗人史诗级buff演奏模式，作为打本之余的一个不错的消遣，喜欢演奏的玩家千万不可错过。

机工士
− 机工士介绍 ...
人们利用帝国的科技力量在伊修加德设立了魔导火炮研发基地，是3.0新增的物理远程职业，使用火枪作为进攻手段，同时也有一个类似召唤兽的小机器人(虽然召唤不费劲，但是它不能自主移动)，提供一定的团队增强能力。另外它也是唯一一个提供全减伤技能的物理职业。
机工的核心手法是控制枪管温度和特殊弹的装填，最大化利用爆发期的野火打出超高的伤害，同时利用其它技能辅助稳定非爆发期的平稳输出。机工的枪管温度控制相较于其它职业更复杂困难一些，而且野火所占输出伤害比例巨大，所以机工士通常需要利用一切可以利用的力量为自己的10s代言，因而如果想要成为最顶尖机工的话，就非常依赖团队的配合。而机工本身的循环经过练习形成肌肉记忆后，相较而言变数不会非常大，算是一个打得多了才能流畅的职业。
目前优秀的机工可以处在输出榜的第一梯队中，和近战分一杯羹，推荐喜欢挑战难度，追求操作难度和精细度的玩家。另外机工士的职业战斗动作也是出了名的帅气~喜欢机械质感和枪弹的玩家也不要错过了。

远程魔法
远程魔法就是需要通过读条进行魔法输出的职业，每个魔法职业的输出核心都完全不同。另外魔法职业的技能特效也都格外惹眼(近战贼气)。
如果你喜欢读条又不想玩奶妈，或者想时刻看到绚烂的技能特效，那就要选魔法职业了。
3个魔法职业和远程物理一起抢夺高难度8人团队剩下的2个远程位置。
黑魔法师 (咒术师)
− 黑魔法师介绍 ...
在乌尔达哈钻研毁灭之力的咒术师最终成为了黑魔法师，用心中的意志毁灭一切阻拦。
黑魔法师是曾经的AOE王者，如今的人形死也不走炮台。使用冰火雷三种属性的魔法进行攻击，其中雷法是DOT型伤害，火系魔法提供极高的伤害，并消耗大量蓝，冰系魔法伤害相对较低，但能让黑魔法师迅速回蓝。因此黑魔可以说是永不缺蓝的永动机。
天语是黑魔的持续性增伤buff，黑魔的核心就是在保持天语不断的前提下，保持雷dot并循环冰火buff，以此打出巨量输出。
黑魔的输出技能全部需要读条(有个不痛不痒的瞬发技能，没啥人用)，加上地面放置型buff黑魔纹加持，移动战对于黑魔来说是非常难以接受的，很多伏地魔们宁愿死在黑魔纹里而被奶妈视为眼中钉，同时新手也经常在移动和处理机制的过程中断天语甚至断了命。对于黑魔法师来说，熟悉副本时间轴与机制是重中之重，使用以太步和魔纹步进行移动，而一旦成为优秀的黑魔，自然就能成为第一梯队的输出选手。

召唤师 (秘术师)
− 召唤师介绍 ...
在利姆萨罗敏萨成为秘术师之后，经历与蛮神的战斗，就能够接触到第三星历诞生于美拉西迪亚大陆的职业——召唤师，它是现在玩家能玩到的(设定上)最古老的职业。通过召唤蛮神之灵与自己并肩战斗，使用DOT进行持续性伤害，并保有极强的机动性。
召唤的主要输出依靠DOT、以太能力技能及龙神附体带来的伤害加成，在龙神期间打出极高的爆发伤害。召唤本身不存在连击系统，大伤害技能大都不需要读条，因此召唤技能自由度相对较高，爆发循环又非常稳定，这为他的机动性提供了良好的保障。虽然木桩输出不如黑魔，但在移动较多的大型副本中，召唤仍旧能打出比黑魔更高的输出。
在3.X版本中，召唤极强的AOE能力让它成为了4人副本快速通关的必备配置，而在如今的版本中，召唤不再有那么亮眼的AOE输出，但仍略高于其他职业。
在8人高难度副本中，召唤目前以较高的输出、灵活的走位和强大的团辅(复活)独领风骚。而召唤独特的召唤兽系统和可以召唤灵体巴哈姆特的特性，更是让不少喜爱召唤兽陪伴的玩家割舍不下。

赤魔法师
− 赤魔法师介绍 ...
4.0的新增职业，设定上是黑魔法师和白魔法师在逃难时交流融合，共同创造的新魔法体系，因此赤魔可以使用黑魔法的火雷核爆，以及白魔法的风土神圣，使用对应魔法技能的时候能够上升黑白魔元的值，通过平衡增加黑白魔元，令自身的武器技能变为魔法攻击，并突刺进行近战攻击。只要玩家拥有任意一个50级战斗职业就可以去乌尔达哈转职成为赤魔法师。
赤魔拥有独特的连续咏唱特性，使用一个读条技能之后的技能不再需要读条。赤魔利用这样的特性快速积攒黑白魔，并打出自己的输出循环。同时因为连续咏唱特性，很多赤魔都非常容易开场OT。
对于新人来说，赤魔是最容易上手的DPS职业没有之一，60级之前甚至没有几个DPS能打得过赤魔，然而满级之后情况就截然不同了。目前而言赤魔的输出能力是3名法系DPS中最低的，但是它拥有使用智力计算威力的治疗技能，和迅速连续复活的能力，在高难度副本开荒时，是重要的保障性角色。
战斗职业的选择
如果看完了上面一大堆你还没有头绪，我简明扼要地帮助你快速选择职业…
如果你没有任何MMORPG的经验，又觉得自己不打算认真研究，仅仅是想随便体验这个游戏，推荐使用秘术师进入游戏(并在后期转为召唤师)，或者使用激活直升包后，在乌尔达哈转职为赤魔法师；
如果你喜欢贴身战斗的感觉，那么早期的枪术师比较适合初学者，而格斗家的打击感更强，当然也需要更多操作；但是从满级状态而言，武僧的难度要比龙骑低一些；
如果你曾经玩过MMORPG，并且了解T奶的职责；或者虽然没有MMORPG经验，但是有信心挑战T奶职业，那么建议从骑士(T，剑术师)和白魔(奶，幻术师)开始玩起。
如果你想直接玩武士/赤魔/黑骑/机工/占星，那么你需要任意一个职业达到50级，并且主线任务能够抵达对应主城。
+ 还是没头绪，请看官方职业自测量表 ...

当然如果你看了前面的介绍，有了自己的想法(帅才是最重要的)，放心去体验，海德林的座右铭是——倾听，感受，思考
战斗职业的强弱、人气和需求
所有职业能顺利完成主线，而且没有太大强弱差异(野外1v2都容易死，无论T奶D)，打怪速度也基本差不多(近战DPS相对快1~2个技能)。
DPS强度从木桩表现来说，黑魔和武士独领风骚。在副本表现中，在不考虑团辅的情况下，DPS通常是武士>武僧>其他DPS，但是在有龙骑忍者的情况下，机工的DPS是可以跟武僧抗衡的。具体DPS排名和表现则会根据副本机制和节奏发生变化。
然而FF14的职业需求相对弱于手法，一个DPS 4000的龙骑和一个DPS 4000的武士，大家一定会选择前者。每个职业都有所谓的上下限，也就是说，当手法不成熟、不合格的时候，某个职业的下限会非常低，这里以武僧，诗人，黑魔、召唤为下限极低的典型代表。相对来说武士、赤魔的下限并不会那么吓人。不想练习手法，或不擅长某一职业手法的，建议换职业，或避免挑战当前版本的8人RAID副本(或者自己组个休闲娱乐队，能开心也好啊)。
从需求来说，不考虑高难度团本，没有任何职业差别或歧视。在高难度的8人raid中，忍者几乎是必带，龙骑、武僧、武士会带1~2个，机工诗人二选一，法系基本召唤居多，有实力的黑魔也OK，赤魔在进度团中比较少，但休闲团带赤魔的很多。
治疗中，学者是铁饭碗，白魔和占星选1，进度团中会有夜占+白魔的打法。3T较为均衡，一般会根据副本选择(建议T玩家三修，或至少双修)。
