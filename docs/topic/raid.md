# 挑战副本

对于大多数喜爱PVE的玩家来说，打Raid拿装备一直是玩游戏的重要意义之一。FF14的游戏理念一直是缩小玩家版本追赶的差距，也因此游戏副本的设计层次分明，多样的副本选择，适合从家庭主妇到硬核玩家各个层次。

::: segmennt orange

### 挑战副本分类

* 极蛮神：每个版本的先头兵，通常提供性能优秀的武器，帮助玩家开荒零式。另外还会掉落七彩坐骑，集齐同版本七彩坐骑之后，还可以召唤~~神龙~~合体坐骑。
* 零式Raid：偶数版本的重头戏。FF14是每2个小版本更新一次装备上限，因此打当前版本对应的零式Raid，是获得当前版本毕业装备的主要手段之一。
  * 普通Raid：零式Raid的前置，难度相对比较低，是比较轻松获得初期装备的方式。
* 24人副本：奇数版本的24人大型迷宫，有着优秀的故事剧情（5.0的24人副本是与尼尔系列联动的），同时提供中间装备，供没有挑战零式的玩家进行过渡。
* 绝境战：奇数版本的最难副本，是最顶尖的玩家才能够挑战的最强敌人。

## 红莲狂潮（4.0）挑战副本

如果你想要挑战本版本的高难度副本，只有成功挑战RAID副本(零式)的玩家才能获得版本的毕业装备。

### 极蛮神：东方圣兽

位于远东之国的蛮族召唤了须佐之男和 舞法 吉祥天女，等待冒险者的挑战。逃窜而去的神龙和主线中的另一位被召唤出的蛮神，也在异国的吉田的诗歌下化为更强大的敌人重新立于冒险者们面前。
4.x的另一条蛮神故事线是四圣兽奇谈，目前有白虎、朱雀和青龙。
4.x极蛮神也会掉落坐骑，这次的坐骑是各色狗子(当然也会飞了)，集齐7条狗子可以召唤出九尾(玉藻御前)。

这些蛮神每一个都有不同于以前蛮神、BOSS的新机制，无论是新玩家还是老玩家都会遇到难以应付的困难点，十分有意思。

### 零式Raid：欧米茄

欧米茄最早在2.3主线中就被提及，它是亚拉戈帝国创造的用来制御巴哈姆特的武器，而在3.57中为了抵抗神龙而被释放。
从神拳痕接任务打开欧米茄的大门，在各个空间中与欧米茄创造的4个小兵战斗。完成每个阶段普通欧米茄的4个副本之后，就可以在神拳痕开放零式欧米茄副本。
零式德尔塔(O1s~O4s)掉落340装备/345武器，零式西格玛(O5s~O8s)掉落370装等装备/375武器，零式阿尔法(O9s~O12s)掉落400装等装备/405武器，是4.x版本的最高装等。

### 24人副本：重返伊瓦利斯 ...

伊瓦利斯是FF12及之后FFT一系列游戏的世界，作为4.X系列的24人大型副本，重返伊瓦利斯的剧本也是由伊瓦利斯的创造者——松野泰己负责的，另外部分画面设计则是由雨宮慶太负责。如果是有玩过FF12或FFT的玩家，应该会对这一系列副本的剧情、场景乃至人物、服装都产生相当熟悉的感觉。就算你没有接触过前面说的这两部作品，这次的24人副本也会为你带来全新的体验，无论是场景、技能特效还是副本机制都会为你带来全新的感觉。
重返伊利瓦斯第一弹是“失落之都拉巴纳斯塔”，需要从黄金港接一系列任务，登上来自帝国的剧团飞艇，探索伊利瓦斯的故事。第二弹是“封闭圣塔黎铎拉纳大灯塔”，第三弹是“欲望之所瓯博讷修道院”。

### 绝境战：绝巴哈 & 绝神兵

从设定上来说，这绝不是随随便便就可以去挑战的内容，但如果你对最高难度的挑战充满热情和期待，那就绝对绝对绝对不要错过了！进入绝系列副本的基本要求是当版本Raid全通(对于绝巴哈来说是通过O4s，对于绝神兵来说是通过O8s)，由于绝系列暂时不开放等级同步，导致如果想要去开荒绝系列，必须有对应版本要求的毕业装(绝巴哈装等限制为345，绝神兵则限制在375，如果你穿着370毕业装去打绝巴哈的话，是根本没办法打的。部分玩家会选择优雷卡升级的345校服代替原本的340毕业装去打绝巴哈)，光这两条要求就会筛掉一大批挑战者。

4.1第一弹为巴哈姆特绝境战，吉田本来放话说如果3周无人攻略就降低难度，结果在开放15天后出现了首杀队伍(当年巴哈首杀的L团)。当然对于其他跃跃欲试但又有些担忧的玩家来说，吉田也表示过，如果P1都打不过去就不用再打了(P1时间为3min)。
绝巴哈需要以击破O4s作为前置，在黄金港异国的吉田对话开启。绝巴哈战斗全程持续大约20分钟，需要8名玩家齐心协力迈过一层层考验，每个技能都严重考验着玩家对于细节的把控和处理。因为挑战绝巴哈而产生分歧、争执乃至心灰意冷的团队和玩家绝不在少数，但也因此击破黄金大肥龙的瞬间更显得难能可贵。

因为绝巴哈太过困难，导致绝巴哈的武器和称号更接近于一种精神象征，在各个交流平台都会遇到玩家与rmt玩家之间的剧烈冲突，可以说黄金武器代表着不可回避的重责，全游戏玩家的双眼都会紧盯不放。
HINT：如果有完整打过巴哈姆特(T5、T9、T13)的玩家，对于绝巴哈的部分机制会比较容易处理。

4.3的绝境战是究极神兵，作为2.0第一个高难度8人讨伐战，纠结神兵融合了风火土三蛮神的技能并增加了额外的机制。在国际服开荒时出现了P5机制需要P1~P3机制特殊处理的意外状况，导致很多进度开荒团不得不退回P1重新开荒。国际服挑战首杀的多个团队都表示这种与设计组斗智斗勇的解谜过程，为绝本的开荒带来了无与伦比的乐趣。
整体来说，在有成熟攻略的现在，绝神兵的难度相较绝巴哈有所降低。

## 苍穹禁城（3.0）挑战副本

………这些副本需要完成对应的60级主线任务。其中蛮神本掉落蛮神鸟和武器，24人副本掉落装备宠物和家具材料，另外还有真正的RAID——亚历山大机神城。
对难度副本感兴趣的玩家推荐最低装等同步并点掉超越的A3S和A8S。

### 极蛮神：上古蛮神
在丧灵钟的石之家，我们会接到名为“上古众神”的任务，这里的上古众神其实指的是被亚拉戈帝国封印的古代蛮神。
但其实在这个任务串里，除了古代蛮神，我们还会遇到库尔扎斯、龙堡和阿巴拉提亚的子民们召唤的新蛮神。
其中云神俾斯麦、武神罗波那、魔神萨菲罗特、女神索菲亚和鬼神祖尔宛的歼殛战都是在这里开启的，而邪龙(尼德霍格传奇征龙战)和骑神(圆桌骑士幻想歼灭战)则需要在摩杜纳，由爱幻想的诗人为大家开启…

3.X一共有7个极蛮神(云、武、骑、邪龙、魔、女、鬼)，会掉落7种蛮神鸟，集齐7种蛮神鸟，在田园郡想要带路的 骗子 领路人那里就可以换到凤凰。
另外3.X的极蛮神也为黑黑的非洲冒险者准备了低保，只要99个极蛮神掉落的低保就可以换到一条鸟~！blink
10个蛮神低保也可以换到该蛮神的武器(骑士剑是7个，盾3个)。

### 24人副本：玛哈之影
在伊修加德砥柱层可以接到任务“空贼登场”。在这里我们会遇到空贼和他的小伙伴们，这串任务为我们带来玛哈之所以覆灭的历史。
依次击破“魔航船虚无方舟”、“禁忌城邦玛哈”和“影之国”，彻底击溃妖异的野心，并见证空贼之间的情谊。

每次完成魔航船、玛哈和影之国时，会分别掉落一枚(不同的)大钱币，集齐多款钱币就可以兑换材料，把260装备升级为270。
玛哈系列副本都有一定的机制需要处理，对T、奶、DPS都分别有所要求，进本之前务必看攻略(除非你组了24个想一起从零开荒的基友…)。

### 零式Raid：亚历山大机神城
亚历山大就是3.X版本的RAID副本，分为启动之章(A1~4)、律动之章(A5~8)和天动之章(A9~12)三个阶段。而无论是启动、律动还是天动又都分为普通亚历山大和零式亚历山大。
普通亚历山大需要在田园郡接任务“迈向毁灭”，然后需要依次挑战亚历山大启动之章1234完成任务，然后继续任务则会开启律动之章和天动之章。

而零式亚历山大则需要在完成普通亚历山大之后，去摩杜纳找异国的诗人分别接任务“ 英雄叙事诗：机神城之章”(如果没有这个任务的话，可能需要先接“英雄叙事诗：幻想之章”，并完成 纠结 究极神兵的讨伐才可以)、“ 英雄梦想诗：机神城之章”、“ 英雄幻想诗：机神城之章”开启。
在绝系列登场之前，最难的RAID副本当属零式亚历山大机神城律动之章(A8s)，在英雄榜截止更新之前，全区全服只有1880个账号通过了A8s，而通过了天动之章A12s的账号一共是9103个。也是因为律动难度过高导致挑战玩家减少，继而天动篇难度下调，并在之后追加了绝系列副本。

## 新生（2.0）挑战副本

这些副本只需要完成50级主线任务就可以打开，现在已经不属于高难度的副本，如果你需要蛮神马可以继续去挑战(但是难易度降低了很多)，另外制作魂武的玩家可能需要打对应的24人副本周常，而RAID本几乎只有为了幻化外观和乐谱的玩家会去逛一圈。

另外想要体验难度的玩家，推荐最低装等同步并且点掉超越之力的JTT、T9和零式巴哈系列。

### 极蛮神：蛮神真正的力量
在沙之家于里昂热那里接到任务“蛮神真正的力量”，可以打开蛮神讨伐的任务串。
首先接到的火风土神的讨伐战，这一系列被称为：真火、真风、真泰坦(ZTT)；以和之后的歼殛战区分。
完成后会打开“蛮神的极限力量”，先会挑战极风神、极泰坦(JTT、蒋婷婷)、极火神。然后根据50级主线进度会逐渐开启极贤王、极水神、极雷神、极冰神的副本。
元素蛮神都会有一定概率掉落蛮神马，集齐6匹蛮神马可以在摩杜纳的“异国的诗人”那里换到麒麟。(其中土风火还会掉落另一种黑色独角兽——梦魇)

### 24人副本：水晶塔
在摩杜纳接到任务“古代亚拉戈文明的遗产”，并完成一系列任务串之后，就能开启24人副本——水晶塔。
水晶塔分为3个阶段(副本)——“古代人迷宫”、“希尔科斯塔”和“暗之世界”。
这条故事线介绍了亚拉戈帝国的历史和他们的科技力量，以及关于第七灵灾背后的一些故事。
依次挑战完这三个副本之后，可以在摩杜纳开启水晶塔周常(每周接一次)“大魔法师的祝福”，完成周常任务之后，可以获得制作魂武所需的“亚拉戈绝灵油”。

古代人迷宫有一些机制杀，如果团队内有大佬指挥的话请听指挥，如果没有的话记得看一眼攻略。

### Raid副本：巴哈姆特
完成真泰坦的讨伐之后，沙之家会追加“蛮神苏醒”的任务。这就是50级8人RAID副本——巴哈姆特的开启任务。
如果你有看过2.X的开篇动画，可能对那头从红色卫月中出现的巨大肥龙印象深刻，2.X的RAID副本就是探索掉落卫月，并最终找到巴哈姆特的故事。同时它在故事情节设计上也是承上启下的副本，壮大的战斗场景和绝美的战斗BGM让很多玩家赞不绝口。
巴哈姆特大迷宫分为3个阶段，分别是邂逅之章(5)、入侵之章(4)和真源之章(4)。最初的任务需要打过邂逅之章5(T5)才可以继续进行，并开启入侵之章，完成入侵之章4(T9)之后，可以继续任务开启真源之章，在真源之章4(T13)中，我们最终会面对巴哈姆特的本尊，并彻底击败它。

由于现在可以解除等级同步进行战斗，巴哈姆特大迷宫不再难以企及，除了T5、T9之外，就算是萌新(60级)也可以比较轻松地打过。
另外完成巴哈姆特大迷宫任务线之后，可以在摩杜纳接到“英雄叙事诗：幻想之章”的任务，挑战 纠结 究极神兵之后，就能开启零式巴哈。
零式巴哈作为2.5版本末期最后一个挑战副本，是T6~T9的加强加难版，当年能通关的队伍屈指可数，而现在就算是解除同步也不是那么容易通关的。如果实在无所事事，也可以挑战看看。
